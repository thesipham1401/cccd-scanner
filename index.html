How to launch

One repo, three paths, same app.
localhost:3000 / Vercel / production
LocalServe the repo root, then open `/` for this hub or `/cccdscanner/` for the scanner.
PreviewImport the repo into Vercel and use the preview URL to test camera and QR.
ProductionKeep Apps Script deployed, then share the Vercel URL with users.
